## 🚀 Overview

An interactive **Employee Performance Dashboard** built with **Streamlit** that analyzes workforce productivity using key metrics such as:

* 📌 Projects Completed
* ⏱ Hours Worked
* 📈 Performance Score
* ⚡ Efficiency

The app provides **real-time insights, KPIs, and visualizations** to evaluate employee and departmental performance.

---

## ✨ Features

* 📂 Auto-load dataset from repository
* 📤 Optional CSV upload support
* 📊 Department-level analytics
* 👤 Employee-level performance tracking
* 📌 Key performance insights (KPIs)
* 📈 Interactive charts using Streamlit

  * Efficiency = Projects Completed / Hours Worked

## 📥 Dataset

The app expects a dataset (`employees.csv`) with columns:

* employee
* department
* projects_completed
* hours_worked
* performance_score

You can:

* Use the default dataset in the repo
* Or upload your own CSV
